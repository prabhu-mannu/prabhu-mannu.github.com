Can monitor the cpu and disk temperatures  using `senors` command.

In *redhat 7* and *centos 7*  have to install **lm_sensors** package.

{% highlight shell %}
yum reinstall lm_sensors -y
{% endhighlight %}

#### Example output :

{% highlight shell %}
